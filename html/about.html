SuperTech is an e-commerce
website that provides customers
the latest tech in mobile
accessories.

- Our Website -

Super Tech is a website to buy phone accessories such as phone cases and headphones, but there are other tech-related products that are available. The website is designed to allow the user to navigate the site with ease, taking inspiration from various websites and using colours effectively. The colour scheme of varying shades of the colours black and white serves to highlight elements of the site and make it easier for users to navigate. The gradual contrast between the shades makes the site more appealing to the eye. There are five pages in total, most of the effort has been put to the “Catalogue” page as it’s where the users will do their shopping. It has sorting tools such as refinement on the left side of the catalogue page that would help users find the item they want, including a drop-down menu when the users hover on the “Catalogue” link on the navigation bar.

The server-side scripts are written in PHP while the browser data where the customer logs in is stored using HTML session storage. This is because it is easier to implement and saves the user data. The continuous data such as the customer tracking, customer orders, and products is stored on the server using MongoDB. Ajax will be used to connect the client and server. It will keep the basket and stock information on the website coordinated.
